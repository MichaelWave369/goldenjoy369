# GoldenJoy369 First Session

Use this walkthrough to create the first evidence artifact.

## Goal

Seal one real creative round inside the runtime, then begin a return round to prove the No-Zero loop.

## Steps

1. Open the GoldenJoy369 app.
2. Use the starter lens: **GoldenJoy369 Runtime**.
3. Review the starter Spark and Essence.
4. Adjust the Joy Signal honestly.
5. Confirm Round Completion is above `0.72`.
6. Click **Generate SealPacket**.
7. Confirm the lens enters the Seed Vault.
8. Add one SeedNote.
9. Click **Begin Return Round**.
10. Fill the return round with the next smallest build action.
11. Export JSON.

## First non-self-test batch

After the self-test, log 3–4 real lenses:

- PhiOffice369.
- Professor Phi.
- ShadowSelf369 / PixelForge.
- FrontPorchAI / LumenBridge.

The point is not to finish them. The point is to see whether the Joy Signal and Round Completion score tell you something useful.

## Evidence boundary

The first export is descriptive evidence only. It proves the runtime can log the loop. It does not yet prove that sealing improves re-entry or project quality.
